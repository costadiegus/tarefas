INTRODUÇÃO
A aplicação foi desenvolvida com a arquitetura abaixo descrita:
- Front-end: Foi utilizado a versão 1.6.2 do AngularJS
- Back-end: Java Servlets com JDK 1.8.2
- Banco de Dados: MySQL versão 10.1.25
- Servidor de Aplicação: Tomcat versão 7.0.26

INSTALAÇÃO

Banco de Dados
Toda a estrutura e dados iniciais constam no arquivo tarefas_tarefa.sql na pasta https://github.com/costadiegus/tarefas/sql.

Aplicação
Deve ser realizado deploy do arquivo WAR (Tarefas.war) localizado em https://github.com/costadiegus/tarefas/tree/master/deploy.

Acesso à aplicação
A página inicial da aplicação pode ser acessada pelo acesso à raiz da URL gerada pelo deploy (Ex: http://localhost:8080/Tarefas/)
